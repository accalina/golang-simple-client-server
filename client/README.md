
# Client

### instalation
- edit the "<codename>" to unique code to distinguish the client report on server page. example: testrun
- edit the "<serverip>" to deployed server. format: http://<ip>:<port>
- compile the code with: 
```sh
$ go build .
```
- change the permission: 
```sh
$ chmod +x ./main
```
- run the client: 
```sh
$ ./main
```
