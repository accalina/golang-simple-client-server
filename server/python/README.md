
# Test Server

This folder supposed to be internal test server, for release server please go server/go

### instalation
- create the virtual environtment
```sh
$ virtualenv env
```
- activate the env
```sh
$ source env/bin/activate
```
- intall the required lib
```sh
$ pip install -r requirements.txt
```
- run the server 
```sh
$ python main.py
```
